Trabalhando com listas

Listas ordenadas:

Para a criação de listas ordenadas usamos a tag OL jundo da tag LI em cada item da lista, para a criação de varias LI podemos usar li*+número de vezes que você quer que repita li:
  1. Acordar
  2. Mandar mensagem pro Pauo
  3. Tomar café
  4. Estudar
  5. Praticar
  6. Passar o resto do dia de boa

Para a alteração de listas podemos usar a tag TYPE dentro da tag OL, assims podemons personalizar a nossa lista com "1", "A", "a", "I" e "i" (mas tome cuidado, pois essa tag TYPE dentro da tag OL pode estar vindo a ser obsoleta)

  1. Acordar
  2. Mandar mensagem pro Pauo
  3. Tomar café
  4. Estudar
  5. Praticar
  6. Passar o resto do dia de boa

Uma outra forma de personalizar a lista é indo em HEAD e usando a tag SYTLE e dentro dela usando a tag OL{LIST-STYLE-TYPE: *varias variações*; (Essa é a melhor forma pois não correr o risco de ficar obsoleta tão cedo, a prioriade da personalização é sempre da tag sttle.)

Listas não-ordenadas:

Para a criação de listas não-ordenadas usamos a tag UL jundo da tag LI em cada item da lista:

  • Pão
  • Maça
  • Arroz
  • Feijão
  • Batata

Podemos alterar a personalização da lista usando a tag TYPE dentro da tag UL, as possiveis alterações são "circle", "square" e "disc". (Tome cuidado pois esse método pode estar obsoleto)

Uma outra forma de personalizar a lista é indo em HEAD e usando a tag SYTLE e dentro dela usando a tag UL{LIST-STYLE-TYPE: "circle", "square" e "disc"; (Essa é a melhor forma pois não correr o risco de ficar obsoleta tão cedo, olhe o HEAD dessa codficação para enteder melhor, a prioriade da personalização é sempre da tag sttle.)

Juntando listas:

É possivel fazer listas dentro de listas da seguinte forma:

Caso queira que o inicio de uma das listas começe pelo final de outra, usa-se a tag START="número que eu queira", assim como é apresentado abaixo:

  1. Carnes:
    1. Frango
    2. Picanha
    3. Carne moida
    4. Hamburguer
  2. Refrescos:
    1. Suco
    2. Refrigerante
    3. Água
    4. Vinho

É possivel também a criação de um terceiro nível:

  • Meu jogos favoritos:
    • Sério Souls
      • Dark souls 3
      • Bloodborne
    • Jogos de RPG
      • Vampry
      • Skyrim
