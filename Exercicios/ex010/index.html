Links para download

    Link de download para PDF não funcionam, isso ocorre porque o navegador já reconhece como pdf e manda para a janela padrão de pdf, mais uma dica é sempre usar a tag TARGET="_BLANK" para abrir o pdf em outra janela, além da tag TYPE="application/pdf".

  • Baixar o livro pdf

    • Para arquivos em zip, é possivel fazer o download com a tag A e selecionando o arquivo requirido, usando também a tag DOWNLOAD com o nome do arquivo e a tag TYPE para indicar ao navegador que existe um arquivo sendo baixado
