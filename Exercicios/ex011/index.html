Imagem dinâmica

Utilizando a tag PICTURE podemos fazer com que as imagens se tornem dinâmicas e se adaptem ao tamanho da tela do usuário, dentro da tag picture usamos podemos usar a tag IMG e a cereja do bolo, a tag SOURCE, nesse caso em específico usamos a tag SOUCE:MEDIA:TYPE, contudo existem outras váriações. Dentro da variavel MEDIA é recomendado sempre usar max-width ao inves de min-width, logo após informar em px (pixels) o tamanho maximo que a imagem não pode ultrapassar para não ser trocada pela imagem menor, como no exemplo abaixo, onde a maior imagem possue 1000x1000 e o seu max-width é de 1050px. Após arrumar essas variaveis existe também a variavel SRCSET, onde nela escolhemos a imagem que ira substuir a imagem grande caso ultrapasse os limites de pixels, é possivel ver um exemplo logo abaixo também, já na variavel TYPE dizemos o tipo de imagem para informar ao navegador o tipo de arquivo que está sendo apresentado, como por exemplo TYPE="IMAGE/PNG ou JPEG".

Tente abrir esse site em varios dispositivos diferentes ou simplesmente aumente e diminua o tamanho do seu navegador.

Reproduzindo Audio

vamos aprender a reproduzir audio!!

Essa é a maneira mais simples de colocar audio, mas também a mais propensa a erros. Usamos a tag AUDIO junto de SRC e nesse caso usamos o parametro CONTROLS para criar uma interface ao usuário para controar o áudio.

Outra forma mais completa de colocar audio

Usando um conversor, podemos alterar o MP3 para WAV ou OGG, é bom usar os três tipos ao mesmo tempo, pois em alguns navegadores, audios em MP3 não carregam automaticamente, ficaria mais ou menos assim:

Caso o audio não tenha aparecido, infelizmente seu navegador não consegue reproduzir áudio. Clique aqui para baixar o arquivo em MP3.

O download não funciona devido ao mesmo erro do pdf, o navegador abre em outra janela ao inves de realizer o download, para resolver esse erro, use TARGET="_BLANK".

Evite sempre usar mídia WAV pois elas são MUITO pesadas.

Para controloar o áudio com maestria, existem alguns parametros que podemos usar, como o parametro AUTOPLAY, que roda o áudio automaticamente, o parametro CONTROLS, que dá o usuário uma interface para controlar o áudio, o parametro LOOP, que faz com que o áudio fique rodando em loop, e o parametro PRELOAD, que possue algumas variaveis, sendo ela AUTO (O site só abrira se o áudio estiver totalmente carregado), METADATA (O site pegara somente algumas informações cruciais sobre o áudio) e por ultimo NONE (O site não pegara nenhuma informação sobre o áudio).

Dentro de AUDIO podemos usar a tag SOURCE SRC para achar o arquivo em mp3, ogg e wav. Assim como no exemplo acima
